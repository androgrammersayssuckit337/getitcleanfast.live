🚀 How to Connect Your Domain (getitcleanfast.live)

  1. Go to Netlify and drag this file to deploy your site
  2. Copy your Netlify site URL (example: yoursite.netlify.app)
  3. Login to Namecheap → Domain List → Manage → Advanced DNS
  4. Add this A Record:
    Host: @ → Value: 75.2.60.5
  5. Add this CNAME:
    Host: www → Value: yoursite.netlify.app
  6. Go back to Netlify → Domain Settings → Add Custom Domain
  7. Enter: getitcleanfast.live
  8. Wait up to 24 hours for DNS to fully connect
